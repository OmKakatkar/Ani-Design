Navigation

Ani-Design provides various classes to quickly style avatars with different shapes and sizes. Style an avatar with class .avatar along with the required classes (e.g. .rounded).

Top Navigation Bar

Navigation bars on the top of a page contain all the links that the user can visit to on your website. They also contain the brand logo and brand name of the website. Use the .fixed class infront of the .nav class to fix it to the top of the page.

Side Navigation

Side navigations contain a list of all the links the user can visit on the site. They generally do not link to an external website. This can be used without the nav HTML tag to convert it into a clickable list.
