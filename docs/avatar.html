Avatar

Ani-Design provides various classes to quickly style avatars with different shapes and sizes. Style an avatar with class .avatar along with the required classes (e.g. .rounded).

Avatar Sizes

Avatars are available in various sizes. Adding a size class e.g. .lg will change the size.

Avatar Shape

Avatar are by default circular. Change the shape by adding .square or .rounded.
